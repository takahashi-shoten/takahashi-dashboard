# 高橋商店 現金管理アプリ

現場で働く高橋商店専用のキャッシュ管理PWAアプリです。
スマホで10秒以内に入力できることを最優先に設計しています。

---

## 起動方法

### ローカルで動かす場合
```bash
# Pythonがある場合
python3 -m http.server 8080

# Node.jsがある場合
npx serve .
```
ブラウザで `http://localhost:8080` を開く。

### GitHub Pagesにデプロイする場合
1. このリポジトリをGitHubにpush
2. Settings → Pages → Source: main branch / root
3. 発行されたURLをブックマーク

---

## iPhoneホーム画面への追加（PWA）

1. Safariでアプリを開く
2. 画面下の「共有ボタン」（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 名前はそのまま「現金管理」→「追加」
5. ホーム画面のアイコンから起動できるようになります

> オフラインでも使えます（一度開いた後）

---

## 基本の使い方

### 口座残高の初期設定（最初に一度だけ）
1. 「設定」タブ → 「口座残高管理」
2. 各口座の「設定」ボタンをタップ
3. 現在の口座残高を入力して保存
4. これ以降の引き出し・入金・移動は自動計算されます

### 現金使用を記録する
- **ワンタップ**: 「よく使うボタン」から分類を選んでタップ
- **音声入力**: マイクボタンを押して「コメリ18500」「嫁に5万」などと話す
- **テキスト入力**: テキスト欄に入力して「自動解析して入力」ボタン

### 銀行から現金を下ろした
「現金引き出し」ボタン → 口座・金額・日付を入力

### 口座への入金
「口座へ入金」ボタン → 売上入金・振込受取などを記録

### 口座間の資金移動
「口座間資金移動」ボタン → 移動元・移動先・金額を選択

## 音声入力の例

| 話す内容 | 認識結果 |
|---------|---------|
| 嫁に5万 | 事業主貸 ¥50,000 |
| コメリ18500 | 材料費 ¥18,500 |
| ENEOS8200 | 車両費 ¥8,200 |
| 鈴木さんに応援代3万 | 外注工賃 ¥30,000 |

| 旭川信金に5万入金 | 旭川信用金庫へ入金 ¥50,000 |
| 北洋から旭川に5万移動 | 口座間移動 ¥50,000 |

## データ保存場所

- **保存先**: このスマホ/PCのブラウザ内（localStorage）
- **サーバー送信なし**: データはデバイスの外に出ません
- **容量**: 通常の使用で数年分のデータを保存可能

---

## CSV出力方法（弥生青色申告用）

1. 「集計」タブを開く
2. 月を選択（◀ ▶ ボタン）
3. 「CSVダウンロード」ボタンをタップ

出力項目:
- 日付 / 借方科目 / 貸方科目 / 金額 / 摘要 / 分類 / 口座 / 領収書有無 / メモ

> 弥生青色申告の「仕訳インポート」機能に合わせた形式で出力されます

---

## バックアップ・復元方法

### バックアップ
設定 → 「バックアップ（JSON）」→ ファイルを保存

### 復元
設定 → 「バックアップから復元」→ 保存したJSONを選択

> 機種変更時はこの手順でデータを移行してください
